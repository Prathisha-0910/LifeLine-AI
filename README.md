# 🚑 LifeLine AI

AI-powered emergency response system that connects users to the nearest hospital and dispatches ambulances intelligently.

---

## 🚨 Problem
In emergency situations, delays in finding hospitals and ambulances can cost lives.

---

## 💡 Solution
LifeLine AI:
- Detects emergency type
- Finds nearest hospital
- Assigns ambulance
- Provides ETA and live map

---

## ⚙️ Features
- 📍 Location-based hospital detection
- 🚑 Smart ambulance allocation
- ⚡ Priority-based emergency handling
- 🗺 Map integration

---

## 🛠 Tech Stack
- FastAPI (Backend)
- Next.js (Frontend)
- n8n (Workflow automation)
- Google Maps API
