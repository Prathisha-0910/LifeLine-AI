# 🚑 LifeLine AI – Emergency Response Orchestrator

## 📌 Problem Statement
In emergency situations, critical time is lost due to lack of real-time coordination between hospitals, ambulances, and emergency contacts.

---

## 💡 Solution
LifeLine AI is a multi-agent system that automates emergency response by:
- Detecting location
- Finding nearest hospitals
- Assigning ambulances
- Calculating fastest routes
- Notifying emergency contacts

---

## 🧠 Architecture
- Backend: FastAPI
- Agents:
  - Planner Agent
  - Location Agent
  - Hospital Agent
  - Route Agent
  - Ambulance Agent
  - Notification Agent
- Workflow: n8n

---

## ⚙️ Tech Stack
- Python (FastAPI)
- n8n (workflow automation)
- React (frontend)
- APIs (Maps, Messaging)

---

## 🚀 Features
- Multi-agent decision making
- Real-time emergency coordination
- API orchestration
- Automated notifications
